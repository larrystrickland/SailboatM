Loading Files

To load files:

  1. Click on "Choose Files" button, to open teh file browser
  2. Select the files you want to open
  3. Click on the "Open" button to close the dialog
  4. Click confirm to load the files

Plotted Objects

Plotted objects include Boats, Marks and Lines

Boats:Boat paths are loaded by loading a file (format needs to be specified)

Marks:To create a mark, simply click the fight mouse button on the map

LinesTo create a mark, simply hold down the right mouse key drag the mouse from the start to the end of the line

Managing Plotted objects

Change label:to rename an object, click on the icon, edit the label and select "Update Label" button

Change color:to change the color of an object, click on the current color and select a color from the list of options

Hide boat path:to hide a boat path, click on the icon, and that path will be hidden

Show boat path:to show a hidden boat path, click on the icon, and that path will be shown

Rotate View

To rotate the view, drag the rotate view slider. This can be very slow if lots of data is loaded. Ideally, just load a small amount of data, orient the view, then load the rest of the data

Annimate

The boats paths can be annimated, but adjusting the start and finish times on the bottom slider, then clicking on start annimation button

The textbox has a time multiplier in it, increaseing this speeds up the annimation, making it negative, causes the boat paths to go backwards.

Saving configuations

You can save the configurations (of boats, marks and lines) by clicking on Save and saving the json file

Loading configuations

You can load a previously saved configurations (of boats, marks and lines) by clicking on Load and selecting the previously saved json file
